# UML Diagrams for Crop Yield Prediction System

This folder contains all UML diagrams for the Crop Yield Prediction System.

## Diagrams Included

1. **Class Diagram** (`class_diagram.puml`) - System architecture and class relationships
2. **Sequence Diagram** (`sequence_diagram.puml`) - Prediction flow from user to ML service
3. **Use Case Diagram** (`use_case_diagram.puml`) - User interactions and system features
4. **Component Diagram** (`component_diagram.puml`) - System components and their relationships
5. **Deployment Diagram** (`deployment_diagram.puml`) - System deployment architecture
6. **Activity Diagram** (`activity_diagram.puml`) - Prediction process workflow
7. **State Diagram** (`state_diagram.puml`) - Application state transitions

## How to View

### Option 1: Online Viewers
- [PlantUML Online Server](http://www.plantuml.com/plantuml/uml/)
- [PlantText](https://www.planttext.com/)

### Option 2: VS Code Extension
Install "PlantUML" extension in VS Code to preview diagrams

### Option 3: Command Line
```bash
# Install PlantUML (requires Java)
brew install plantuml  # macOS
# or download from http://plantuml.com/download

# Generate PNG from PlantUML file
plantuml class_diagram.puml
```

### Option 4: Mermaid (Alternative)
Some diagrams are also available in Mermaid format for GitHub/GitLab rendering.

## Diagram Formats

- **PlantUML (.puml)** - Text-based UML diagrams (recommended)
- **Mermaid (.mmd)** - Alternative format for markdown rendering
